# 🥗 NutriTrack - Calorie Tracker

NutriTrack is a responsive web-based calorie tracking application that helps users monitor their daily calorie intake, track workouts, and visualize their progress.

## ✨ Features

- 👤 User Registration & Login
- 🎯 Personalized Daily Calorie Goal
- 🍽️ Meal Tracking
- 🔥 Exercise & Calorie Burn Tracking
- 📊 7-Day Progress Charts
- 📅 Daily History
- 📱 Responsive Design
- 💾 Local Storage Support

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Chart.js
- Local Storage

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/RamchaRan-0000/Calorie-Tracker-Application.git
   ```
2. Open `calories.html` in your browser.
